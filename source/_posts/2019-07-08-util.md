## 2.实践积累
### 2.1对象结构循环
```
// JS形式
for(let key in obj){
	console.log(key);
	console.log(obj[key]);
}
// JSX形式
{Object.keys(obj).map((key, index) => (
     <ListItem key={`obj${index}`} text={key} extra={obj[key]}/>
))}
```
