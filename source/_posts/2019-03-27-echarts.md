## 1.文档目的
维护一个简单的图表模板,积累常用参数,便于快速配置图表
参考网址:[echarts门户网站](https://echarts.baidu.com/)

## 2.内容梳理
图表基础模板(基于vue):
```
// template
<div ref="chart1" style="width: 100%;height:400px;"></div>
// function
initChart() {
  // 各县剩余贫困户
  let chart1 = echarts.init(this.$refs['chart1']);
  let chartOptions = {
    title: {
      text: '各县剩余贫困户',
      x: 'center',
      textStyle: CHART_OPTIONS.TITLE_STYLE
    },
    color: CHART_OPTIONS.COLORS,
    tooltip: {},
    grid: CHART_OPTIONS.GRID,
    xAxis: {
      axisLabel: CHART_OPTIONS.AXIS_LABEL
    },
    yAxis: {
      data: this.oGxsypkh.key,
      axisLabel: CHART_OPTIONS.AXIS_LABEL
    },
    series: [
      {
        name: '户数',
        type: 'bar',
        data: this.oGxsypkh.value
      }
    ]
  };
  chart1.setOption(chartOptions);
}

```
